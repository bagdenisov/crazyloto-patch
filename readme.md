# Библиотеки и патчер для Лохотронщик: Crazy Loto 4

Данная продукция предназначена для игры в "Лохотронщик: Crazy Loto 4" на 
современных операционных системах без ограничений.

Исправления:
* NoCD патч - теперь не требуется держать смонтированный образ диска.
* Локальные сохранения - игра будет сохранять прогресс рядом с собой.
* Слой совместимости DDrawCompat - стабильная работа на последних Windows.

Преимущества:
* Портативность - игра работает чисто в пределах своего пространства.

## Как собрать проект из исходных кодов
1. Подготовка окружения

Для сборки вам понадобятся компилятор mingw-w64 (для 32-битных систем) и утилита make.

* В Linux (Ubuntu/Debian):

```
sudo apt update
sudo apt install build-essential mingw-w64
```

* В Windows:
Установите [MSYS2](https://www.msys2.org/) и выполните в консоли MSYS2:

```
pacman -S mingw-w64-i686-toolchain make
```

2. Сборка

Откройте терминал в папке с проектом и выполните команду:

```
make
```

После завершения процесса в папке проекта появится директория build/, внутри которой будут лежать готовые библиотеки `_dvapi32.dll`, `_ernel32.dll` и патчер `modding.exe`.

3. Очистка проекта

Чтобы удалить временные объектные файлы и скомпилированные библиотеки, выполните:

```
make clean
```

## Как использовать `modding.exe`
Поместите данный файл рядом с игрой `loto4.exe` и запустите `modding.exe`. Программа должна будет создать `loto4_new.exe`, которая будет готова для работы с библиотеками.
